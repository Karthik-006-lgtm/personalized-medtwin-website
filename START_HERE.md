## ‚ö° Quick Start (3 Steps)

### Step 1: Install Prerequisites

You need three things installed on your computer:

1. **Node.js** - https://nodejs.org/ (download and install)
2. **Python** - https://www.python.org/ (download and install)
3. **MongoDB** - https://www.mongodb.com/try/download/community (download and install)

### Step 2: Install Dependencies

Open terminal/command prompt in the project folder and run:

```bash
npm install
cd backend && npm install && cd ..
cd frontend && npm install && cd ..
cd ml-service && pip install -r requirements.txt && cd ..
```

### Step 3: Start Everything

**Windows Users:** Double-click `START.bat`

**Mac/Linux Users:** 
```bash
chmod +x START.sh
./START.sh
```

**Then open your browser:** http://localhost:5173

## ‚ö†Ô∏è Common Issues & Quick Fixes

### "Cannot connect to MongoDB"
**Fix:** Start MongoDB first
```bash
mongod
```

### "Port already in use"
**Fix:** Close other apps using the port or restart computer

### "Module not found"
**Fix:** Reinstall dependencies
```bash
npm install
cd backend && npm install && cd ..
cd frontend && npm install && cd ..
```

### "Frontend shows blank page"
**Fix:** Make sure all 3 services are running (check terminals)
